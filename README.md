# ShelfTalk AI

Plataforma Full Stack com visão computacional + LLMs para gerar conteúdo otimizado de produtos para e-commerce.

## Stack principal

- Frontend: Next.js 14 (App Router), React 18, TypeScript, Tailwind CSS
- Backend: NestJS, PostgreSQL, Redis, BullMQ, LangChain, GPT-4o-mini
- Worker: serviço Node/Nest separado para jobs pesados
- Infra: Docker, Docker Compose, Railway, Vercel

Em desenvolvimento.
